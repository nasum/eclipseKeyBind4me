# 俺俺 eclipse KeyBind
eclipseのキーバインドをemacs"風"にアレンジ。emacsのキーバインドに全て従っているわけではありません。

## emacs準拠のキーバインド

| 動作          | コマンド            |
|:-------------:|:-------------------:|
|前に進む       | Ctrl + F            |
|後ろに進む     | Ctrl + B            |
|次の列に進む   | Ctrl + N            |
|後ろの列に進む | Ctrl + P            |
|保存           | Ctrl + X , Ctrl + S |
|検索           | Ctrl + S            |
|新規作成       | Ctrl + X, Ctrl + F  |
|切り取り       | Ctrl + Y            |
|一行削除       | Ctrl + K            |
|全て選択       | Ctrl + X H          |

## emacs準拠じゃないけどemacsに準拠したら競合しちゃったやつの新しいキーバインド

|動作           | コマンド            |
|:-------------:|:-------------------:|
|やり直し       | Ctrl + Shift + \    |

それ以外はノーサポート。
